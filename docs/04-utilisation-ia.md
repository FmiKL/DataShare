# Utilisation de l'IA dans le développement

Un assistant IA a été utilisé pour la fonctionnalité de suppression des fichiers depuis l'espace utilisateur.

Les commits concernés sont les suivants :

- `feat(ai): add file deletion`
- `fix: improve file deletion`

## Tâches confiées

- ajout d'une route API de suppression
- ajout de l'appel côté frontend
- intégration du bouton de suppression dans la page "Mes fichiers"

## Revue

La première version a été relue puis corrigée sur plusieurs points :

- déplacement de la logique backend dans un service dédié
- recherche du fichier par utilisateur connecté
- suppression du fichier stocké localement
- ajustement de l'interface pour respecter les maquettes
- ajout des tests manquants

Le premier commit correspond à la première version assistée par IA. Le second commit correspond à la revue et aux corrections.
